# DevHub-Colab

A beginner-friendly Python coding curriculum — built for kids learning to code, from pure Python all the way to making real games with pygame.

---

## What's in here?

| Folder              | Topic                                 | Tools       |
| ------------------- | ------------------------------------- | ----------- |
| `00_basics/`        | Variables, loops, lists, functions    | Pure Python |
| `01_turtle/`        | Drawing shapes and patterns           | turtle      |
| `02_pygame_static/` | Windows, shapes, and scenes           | pygame      |
| `03_pygame_moving/` | Animation, keyboard input, math       | pygame      |
| `04_pygame_games/`  | Complete mini-games                   | pygame      |
| `05_challenges/`    | Open-ended projects for fast learners | pygame      |

---

## How to get started

### 1. Install Python

Download from [python.org](https://www.python.org/downloads/) — get Python 3.10 or newer.

### 2. Install pygame

Open a terminal and run:

```
pip install pygame
```

### 3. Run a file

Navigate to any folder and run a script for example:

```
python 00_basics/01_variables.py
python 01_turtle/03_spiral.py
python 02_pygame_static/03_night_scene.py
```

---

## Suggested learning path

1. Work through `00_basics/` to get comfortable with Python.
2. Try `01_turtle/` to see instant visual results.
3. Move into `02_pygame_static/` — open a window, draw shapes, build scenes.
4. Level up with `03_pygame_moving/` — animation and keyboard controls.
5. Play (and study) the games in `04_pygame_games/`.
6. Pick a challenge from `05_challenges/CHALLENGES.md` and build your own!

---

> Tip: Read every file top-to-bottom before running it. The comments explain exactly what each line does.
